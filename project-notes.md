# Todo App Plan

## Tech Stack

- HTML
- CSS
- JavaScript (DOM)

## Goal (Day 1)

- Add task
- Delete task
- Mark task complete

---

## Step 1: HTML Setup

Create:

- Input → id="taskInput"
- Button → id="addBtn"
- List → id="taskList"

---

## Step 2: Select Elements (JS)

Select:

- input
- button
- list

Use:

- document.getElementById()

---

## Step 3: Add Task

When button clicked:

1. Get input value
2. Create `<li>`
3. Set text
4. Append to list

---

## Step 4: Delete Task

Inside each task:

1. Create delete button
2. Add click event
3. Remove parent `<li>`

---

## Step 5: Mark Complete

On clicking task:

- Toggle line-through

Use:

- classList.toggle() or style.textDecoration
